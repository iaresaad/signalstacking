## Product
**Trumpet** (sendtrumpet.com) — a B2B **Digital Sales Room (DSR)** platform; "the intelligent GTM / revenue-collaboration layer." Reps create personalised, trackable buyer workspaces ("Pods") that consolidate the whole deal — demos, decks, proposals, mutual action plans (MAPs), quotes, e-sign, async video/voice — into one shareable link. **Trumpet AI** auto-brands rooms, organises content, surfaces buyer-intent signals, predicts deal health, and recommends next steps.

## Who we sell to (ICP)
- **Economic buyer:** CRO / VP Sales / Head of Revenue. **Influencer:** RevOps, Revenue/Sales Enablement leaders. **Champion / daily user:** AEs, SDRs, AMs, CSMs, Solutions/Sales Engineers.
- **Company type:** B2B (esp. SaaS/tech) with complex, multi-stakeholder sales cycles — SMB through Enterprise.

## Core value props (what to anchor angles on)
1. **Faster cycles & higher win rates** — DSRs cited at ~38% faster cycles, ~21% more closed-won.
2. **Buyer enablement + multithreading** — real-time intent on who's viewing/sharing; surfaces hidden stakeholders; enriches the CRM/forecast.
3. **Consistent, branded, duplicable buying experience** — replaces messy email threads + static PDFs; one hub from first touch → close → onboarding/renewal.

## When Trumpet is a strong fit (relevance hooks → map these to a prospect's signals)
- **Scaling / onboarding many new reps** (hiring AEs/SDRs, new geos) → need a *duplicable, uniform* sales motion fast.
- **Going upmarket / multi-stakeholder enterprise deals** → multithreading + hidden-stakeholder visibility.
- **Weak forecasting / deal visibility** → room analytics + buyer-intent signals into the CRM.
- **Fragmented GTM stack / content chaos / inconsistent brand in deals** → the unifying customer-facing layer.
- **Milestone pressure** (IPO-readiness, ARR/profitability targets, fast growth) → cycle compression + ACV lift strengthens the numbers.
